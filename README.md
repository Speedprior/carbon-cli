# PSCarbonBlackCloud

A set of powershell cmdlets to interract with Carbon Black Cloud.

## Build the Project / Install Dependencies

```console
PS> ./Build-PSCarbonBlackCloud.ps1
```

## Run Tests

```console
PS> ./Invoke-Tests.ps1
```

## Importing the Module

```console
PS> Import-Module ./src/PSCarbonBlackCloud.psm1
```
